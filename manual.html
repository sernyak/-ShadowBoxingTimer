User Manual

Welcome to the Shadow Boxing Timer

Voice Control

"Start"

Starts the workout or the current round.

"Pause"

Pauses the timer.

"Resume"

Resumes the timer after a pause.

"Stop"

Stops the current round, saving your progress.

"Restart"

Restarts the current round from the beginning.

"Reset"

Resets the timer to its initial state.

"Silence" / "That's all"

Turns off voice command listening.

Gestures & Controls

Start/Pause

Tap the large timer display to start or pause.

Open Profile Menu

Swipe left from the right edge of the screen.

Edit Delay / Classic Timer

Long-press the 'Delay' or 'Timer' buttons to change their duration.

Quick Profile Edit

Long-press 'Profile Name', 'Round time', or 'Rest time' to jump directly into the editor.

Quick Select & Exit

In the profile list, tap the timer icon next to a profile to select it and immediately return.

Edit Custom Intervals (Premium)

Long-press any time field in the profile editor to customize intervals for a complex workout.

Finish Session

Long-press the 'Reset' button to end the current workout session and save it to your history.
